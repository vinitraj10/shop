## Overview
Client-Side of the shop project given in Capgemini full stack hackathon challenge

## Project Setup (NodeJs reuired)

<p>Move in Directory</p>

```
cd client
```
<p>Install dependencies</p>

```
npm install

```
<p>Start Server</p>

```
npm run start
```

<p>Access Project</p>
```
http://localhost:8080
```
